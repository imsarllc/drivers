# IMSAR Linux Drivers
**IMSAR's public repo containing GPL licensed sources**

## Contents
### allocated-gpio
Expose GPIO pin controls to user space via sysfs

### intc
IMSAR interrupt controller driver

### jtag
Fork of Xilinx XVC driver

### lcd
Fork of LCD driver for Newhaven NHD-0216K3Z-NSW-BBW

### sarspi
A user space SPI driver that uses `read()`/`write()` instead of `ioctl()`

### uio
IMSAR Memory Mapped Device wrapper
