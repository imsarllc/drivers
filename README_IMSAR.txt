The structure of this repo is based on the structure of the "Driver Package (BSP)"
available from Nvidia here: https://developer.nvidia.com/embedded/jetson-linux-r3521

Please download that file and extract it to this directory.

Some files have been added to support IMSAR specific modifications. In addition,
the sources folder in this repo is populated using Git submodules. The original
Driver Package would populate these folders by using source_sync.sh OR by
downloading the sources from the "Driver Package (BSP) Sources" available on the 
same page.
